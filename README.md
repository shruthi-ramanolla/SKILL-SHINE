# 🌟 Skill Shine

Skill Shine is an interactive learning and skill development platform designed to help users improve their knowledge, explore new skills, and enhance productivity through a simple and user-friendly interface.

---

## 🚀 Features

* 📚 Skill-based learning modules
* 👩‍💻 User-friendly interface
* 🔐 Login and registration system
* 📈 Progress tracking
* 🌐 Responsive web design
* 💡 Clean and simple navigation

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/shruthi-ramanolla/skill-shine.git
```

2. Open the project folder

```bash
cd skill-shine
```

3. Install required dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```
